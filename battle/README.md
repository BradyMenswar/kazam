# kazam-battle

Battle state tracking and domain types for Pokemon Showdown.

## Overview

This crate provides types and logic for tracking Pokemon battle state, including:
- Pokemon stats and details
- Active battle state
- Move and ability tracking
- Team composition

## Status

⚠️ **Experimental**: This crate is in early alpha. APIs are highly unstable and subject to significant changes.

## Features

- `serde`: Enable serde serialization support (optional)

## Usage

```rust
use kazam_battle::*;

// Work with battle state types
```
